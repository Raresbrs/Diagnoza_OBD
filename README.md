# Diagnoza_OBD
Simulare EMS pentru detectarea codurilor de eroare OBD-II. Proiectul generează parametri motor (temperaturi, MAF, ulei, TPS, lambda) și introduce defecte uzuale. Erorile sunt detectate automat și afișate cu explicații. Grafice dedicate arată evoluția parametrilor afectați. Realizat în Python cu NumPy și Matplotlib.
